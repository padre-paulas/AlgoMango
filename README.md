# <img src="./src/assets/mango-logo-2.png" style="height: 40px; width: 40px" />AlgoMango
### AlgoMango is an algorithm visualizer built with **React** and **Tailwind CSS**. It bridges the gap between abstract computer science concepts and visual intuition. 
---
### 🚧 Current Status
- [x] Dynamic Array Generation
- [x] Quicksort Visualization
- [x] Bogo Sort
- [ ] Merge Sort
- [ ] Selection Sort
- [x] Simple and Binary Search
- [ ] Graph Algorithms
- [ ] User System 
---
### 💻 Getting Started
1. Clone the repo:
```
git clone https://github.com/padre-paulas/AlgoMango.git
```
2. Install dependencies:
```
npm install
```
3. Run the development server:
```
npm run dev
```
